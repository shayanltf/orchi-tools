# Library Navigation

Use when choosing an Apple entrypoint.

- SwiftUI UI, Liquid Glass, component patterns, or view refactors -> `swiftui`.
- App Intents, iOS Simulator debugging, simulator inspection, or browser-visible previews -> `ios`.
- AppKit interop, macOS windows, signing, entitlements, packaging, or notarization -> `macos`.
- Xcode build/run/debug, SwiftPM, or test triage -> `build`.
- ETTrace, memgraphs, SwiftUI performance audit, or telemetry -> `performance`.
