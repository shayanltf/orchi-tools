# Voice And Reporting

Use when choosing grammatical person and report density.

## Voice Matrix

- Agent body: first person self-instruction.
- Agent `description`: third person routing signal.
- Skill body: first person or imperative self-instruction.
- Skill `description`: third person routing signal.
- Command body: first person or imperative workflow.
- Command `description`: third person routing signal.
- Context file: third-person declarative; imperative for hard constraints.
- Human docs: third-person declarative.
- Chat to user: second person, concrete, plain.
- Agent-to-agent payload: first person from sender.

## Reporting

- Compressed reports drop filler, not facts.
- Keep identifiers, paths, line numbers, exact errors.
- Security warnings, irreversible actions, and ordered steps stay fully clear.
- Code, docs, configs, and commit messages stay normal prose.
- Handoff docs can be compressed; shipped docs should read cleanly.
